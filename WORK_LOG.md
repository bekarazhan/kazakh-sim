## 2026-06-19 — Улучшение графики (PBR, Постобработка) и коллизий

### Что сделано
- **Коллизии и управление (`FirstPersonControls.ts`, `Overlay.ts`)**:
  - Реализованы коллизии со стенами юрты и возможность прохода через дверь.
  - Исправлено выталкивание игрока и застревание снаружи юрты при обходе стен.
  - Клик по оверлею теперь корректно запрашивает pointer lock на canvas.
- **Окружение и степь (`Steppe.ts`, `Sky.ts`, `World.ts`)**:
  - Создана летняя степь: мягкий рельеф холмов вместо острых гор, добавлено ~700 пучков травы.
  - Настроено яркое летнее солнце (интенсивность 2.8) и мягкий голубой туман (`FogExp2` 60–220м).
- **Материалы, свет и постобработка (`Game.ts`, `Lighting.ts`, `GeomUtils.ts`)**:
  - Переход на `MeshStandardMaterial` (PBR) для всех моделей.
  - Интеграция `THREE.Sky` с физической атмосферой Рэлея (летний полдень, солнце 68°).
  - Использование `PMREMGenerator` для генерации карты окружения (environment map) под PBR.
  - Настройка качественных теней (`4096x4096`, `PCFShadowMap`).
  - Добавлена постобработка через `postprocessing`: `BloomEffect` + `SMAAEffect` + `ACES Filmic ToneMapping`.
  - Устранена проблема z-fighting между полом юрты и ковром (шырдаком).
  - Переход на `THREE.Timer` вместо `THREE.Clock` (убраны deprecation warnings).

### Что работает
- Реалистичное PBR-освещение, динамическая атмосфера неба, тени, bloom-эффект и сглаживание SMAA.
- Коллизия с юртой изнутри и снаружи, корректный проход через дверной проем.

### Что следующее (v1.2)
- Добавить Web Audio API — ветер, треск огня
- Tooltip при взгляде на предмет (raycasting)
- Оптимизация — InstancedMesh для травы
